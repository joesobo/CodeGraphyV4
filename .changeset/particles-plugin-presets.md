Add the first CodeGraphy Particles plugin with Odysseus-style Graph Background presets.

When the plugin is enabled, it adds a Graph Background section to the Theme popup and lets users toggle Synapse, Rain, Constellations, Perlin Flow, Leaves, Sparkles, Embers, and workspace custom TypeScript effects behind the Relationship Graph. The plugin owns the particle controls, canvas renderer, prewarmed animations, green Leaves preset, TypeScript custom effect compilation from `.codegraphy/particles/`, and the repo Fireflies example.

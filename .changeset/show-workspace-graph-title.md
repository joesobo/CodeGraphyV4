Show the current workspace name as the Graph view title and keep graph search controls in a compact strip above the graph.

The Graph view now shows the current workspace name in VS Code's view title, falling back to Graph when no workspace is open. Search, match-case/whole-word/regex toggles, and filter controls stay available while graph data is loading or refreshing without creating a separate blank sidebar pane.

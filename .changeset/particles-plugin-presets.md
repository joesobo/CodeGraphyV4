Add the first CodeGraphy Particles plugin with graph background particle presets.

When the plugin is enabled, it adds a Particles section to the Theme popup and lets users toggle Synapse, Rain, Constellations, Perlin Flow, Leaves, Sparkles, Embers, Snow, and workspace custom TypeScript effects behind the Relationship Graph nodes and edges. The plugin owns the particle controls, canvas renderer, prewarmed animations, green Leaves preset, TypeScript custom effect compilation from `.codegraphy/particles/`, a subtle layered Snow preset, and an orange painterly Fireflies custom effect example with wandering glow, brushy trails, pulses, and flashes under `examples/.codegraphy/particles/`.

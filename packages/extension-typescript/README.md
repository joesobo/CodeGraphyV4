# CodeGraphy TypeScript/JavaScript Plugin

Adds TypeScript and JavaScript dependency analysis to [CodeGraphy](https://marketplace.visualstudio.com/items?itemName=joesobo.codegraphy).

Requires the core CodeGraphy extension: `joesobo.codegraphy`.

Install this plugin alongside the core CodeGraphy extension to detect:

- ES module imports and exports
- re-exports
- dynamic imports
- CommonJS `require()`

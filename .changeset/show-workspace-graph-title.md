Show the current workspace name as the Graph view title and move graph search controls into a dedicated Search view above the graph.

The Search view keeps the search field, match-case/whole-word/regex toggles, and filter controls available while graph data is loading or refreshing. The Graph view now has more room for the graph itself while continuing to react to the same search and filter state.
